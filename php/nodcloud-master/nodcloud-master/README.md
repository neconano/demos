#### :fa-thumb-tack:  介绍
点可云进销存系统，基于thinkphp+layui开发。
功能包含：采购、销售、零售、多仓库管理、财务管理等功能 和超详细的报表功能（采购报表、销售报表、零售报表、仓库报表、资金报表等）

#### :fa-paperclip:  软件架构
thinkphp+layui
